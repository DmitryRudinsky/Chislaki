## Вариант 29: Численное дифференцирование

### Функция
```
y = (2sin(2x) - 3cos(3x)) / √(2x + 3)
```

### Параметры
- **Интервал**: [-1.0, 1.0]
- **Шаг h**: 0.10
- **Шаг h/2**: 0.05

## Реализованные схемы

### Первая производная (6 схем):

1. **Правая разность (2 точки)** - O(h)
   ```
   y'ᵢ ≈ (yᵢ₊₁ - yᵢ) / h
   ```

2. **Левая разность (2 точки)** - O(h)
   ```
   y'ᵢ ≈ (yᵢ - yᵢ₋₁) / h
   ```

3. **Центральная разность (2 точки)** - O(h²)
   ```
   y'ᵢ ≈ (yᵢ₊₁ - yᵢ₋₁) / (2h)
   ```

4. **Правая схема (3 точки)** - O(h²)
   ```
   y'ᵢ ≈ (-3yᵢ + 4yᵢ₊₁ - yᵢ₊₂) / (2h)
   ```

5. **Левая схема (3 точки)** - O(h²)
   ```
   y'ᵢ ≈ (3yᵢ - 4yᵢ₋₁ + yᵢ₋₂) / (2h)
   ```

6. **Центральная схема (4 точки)** - O(h³)
   ```
   y'ᵢ ≈ (-2yᵢ₋₁ - 3yᵢ + 6yᵢ₊₁ - yᵢ₊₂) / (6h)
   ```

### Вторая производная (4 схемы):

1. **Центральная разность (3 точки)** - O(h²)
   ```
   y''ᵢ ≈ (yᵢ₊₁ - 2yᵢ + yᵢ₋₁) / h²
   ```

2. **Правая схема (3 точки)** - O(h)
   ```
   y''ᵢ ≈ (yᵢ - 2yᵢ₊₁ + yᵢ₊₂) / h²
   ```

3. **Левая схема (3 точки)** - O(h)
   ```
   y''ᵢ ≈ (yᵢ₋₂ - 2yᵢ₋₁ + yᵢ) / h²
   ```

4. **Центральная схема (5 точек)** - O(h⁴)
   ```
   y''ᵢ ≈ (-yᵢ₋₂ + 16yᵢ₋₁ - 30yᵢ + 16yᵢ₊₁ - yᵢ₊₂) / (12h²)
   ```

## Формула Рунге-Ромберга

Для точки x = 0.5:

- Центральная разность (h/2): 5.36885994, погрешность: 1.66e-2
- **Рунге-Ромберг**: 5.38542688, погрешность: **6.39e-5** ✨

### Вторая производная

- Центральная разность (h): -5.02285392, погрешность: 3.49e-2
