# Лабораторная работа 3.5 - Численное дифференцирование

## Вариант 29

### Условие задачи

Выполнить сравнение различных схем численного дифференцирования, вычислив значения первой и второй производной для заданной функции y=f(x).

**Функция:**
```
y = (2sin(2x) - 3cos(3x)) / √(2x + 3)
```

**Параметры:**
- Интервал: [-1.0; 1.0]
- Шаг h = 0.10
- Шаг h/2 = 0.05

### Схемы численного дифференцирования

#### Для первой производной (6 схем):

1. **Правая разность (2 точки)** - O(h)
   - y'ᵢ ≈ (yᵢ₊₁ - yᵢ) / h

2. **Левая разность (2 точки)** - O(h)
   - y'ᵢ ≈ (yᵢ - yᵢ₋₁) / h

3. **Центральная разность (2 точки)** - O(h²)
   - y'ᵢ ≈ (yᵢ₊₁ - yᵢ₋₁) / (2h)

4. **Правая схема (3 точки)** - O(h²)
   - y'ᵢ ≈ (-3yᵢ + 4yᵢ₊₁ - yᵢ₊₂) / (2h)

5. **Левая схема (3 точки)** - O(h²)
   - y'ᵢ ≈ (3yᵢ - 4yᵢ₋₁ + yᵢ₋₂) / (2h)

6. **Центральная схема (4 точки)** - O(h³)
   - y'ᵢ ≈ (-2yᵢ₋₁ - 3yᵢ + 6yᵢ₊₁ - yᵢ₊₂) / (6h)

#### Для второй производной (4 схемы):

1. **Центральная разность (3 точки)** - O(h²)
   - y''ᵢ ≈ (yᵢ₊₁ - 2yᵢ + yᵢ₋₁) / h²

2. **Правая схема (3 точки)** - O(h)
   - y''ᵢ ≈ (yᵢ - 2yᵢ₊₁ + yᵢ₊₂) / h²

3. **Левая схема (3 точки)** - O(h)
   - y''ᵢ ≈ (yᵢ₋₂ - 2yᵢ₋₁ + yᵢ) / h²

4. **Центральная схема (5 точек)** - O(h⁴)
   - y''ᵢ ≈ (-yᵢ₋₂ + 16yᵢ₋₁ - 30yᵢ + 16yᵢ₊₁ - yᵢ₊₂) / (12h²)

### Формула Рунге-Ромберга

Для уточнения производных используется формула:
```
y'ᵢ(xᵢ) = φᵢ(xᵢ, h) + [φᵢ(xᵢ, h) - φᵢ(xᵢ, kh)] / (k^p - 1) + O(h^(p+1))
```

где:
- φᵢ - численная схема
- k = 2 (отношение шагов)
- p - порядок точности схемы

### Результаты

Программа выполняет следующие вычисления:

1. **Вычисление производных** для всех точек интервала с шагом h и h/2
2. **Сравнение с аналитическими значениями** производных
3. **Расчёт погрешностей** для каждого метода
4. **Визуализация результатов** на графиках:
   - График первой производной (все схемы)
   - График второй производной (все схемы)
   - Графики погрешностей (логарифмическая шкала)
   - Сравнение шагов h и h/2
   - Таблицы средних погрешностей

### Выводы

1. **Влияние шага**: При уменьшении шага с h до h/2 погрешность уменьшается в соответствии с порядком точности схемы
2. **Порядок точности**: Схемы с большим порядком точности дают меньшую погрешность
3. **Центральные vs односторонние**: Центральные схемы обычно точнее односторонних
4. **Многоточечные схемы**: Схемы на 4-5 точках дают более высокую точность
5. **Формула Рунге-Ромберга**: Позволяет дополнительно уточнить результат
