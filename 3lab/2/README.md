# Лабораторная работа 3.2 - Интерполяция Ньютона

## Описание

Интерполяция функции с помощью многочленов Ньютона 2-й и 3-й степени с использованием разделённых разностей.

## Данные

- **Точка интерполяции:** x* = 0.413
- **Узлы:** 9 точек
- **Степени многочленов:** 2 и 3
- **Оценка погрешности:** метод разделённых разностей

## Запуск

### Графический интерфейс (Electron)
```bash
npm install
npm start
```

### Консольные вычисления
```bash
npm run calc
```

## Функции

- `chooseNearestNodes()` - выбор ближайших узлов
- `dividedDifferences()` - вычисление разделённых разностей
- `evaluateNewton()` - вычисление значения многочлена Ньютона
- `estimateError()` - оценка погрешности интерполяции
- `verifyAtAllNodes()` - проверка интерполяции

## Результаты

Программа выводит:
- Коэффициенты разделённых разностей
- Многочлены Ньютона для степеней 2 и 3
- Значения в точке x*
- Оценку погрешности интерполяции
- Проверку в узловых точках
- Графики интерполяции и погрешности
