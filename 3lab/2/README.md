# Лабораторная работа 3.2 - Интерполяция Ньютона

## Описание

Интерполяция функции с помощью многочленов Ньютона 2-й и 3-й степени с использованием разделённых разностей.

**ОБНОВЛЕНИЕ:** Теперь программа рассматривает **все возможные последовательные интервалы**, а не только ближайшие точки!

## Данные

- **Точка интерполяции:** x* = 0.413
- **Узлы:** 9 точек
- **Степени многочленов:** 2 и 3
- **Интервалов для P₂:** 7 (из них 2 содержат x*)
- **Интервалов для P₃:** 6 (из них 3 содержат x*)
- **Оценка погрешности:** метод разделённых разностей

## Функции

- `chooseNearestNodes()` - выбор ближайших узлов (старый подход)
- `getAllConsecutiveIntervals()` - **НОВАЯ**: генерация всех последовательных интервалов
- `isPointInInterval()` - **НОВАЯ**: проверка принадлежности точки интервалу
- `dividedDifferences()` - вычисление разделённых разностей
- `evaluateNewton()` - вычисление значения многочлена Ньютона
- `estimateError()` - оценка погрешности интерполяции
- `verifyAtAllNodes()` - проверка интерполяции

## Ключевые результаты

### Многочлен 2-й степени
Значения совпадают с методом Лагранжа (как и должно быть):
- Интервал [-0.16, 0.62]: P₂(0.413) = **4.137985**
- Интервал [0.23, 1.01]: P₂(0.413) = **5.177347**

### Многочлен 3-й степени
Значения совпадают с методом Лагранжа:
- Интервал [-0.55, 0.62]: P₃(0.413) = **3.532009**
- Интервал [-0.16, 1.01]: P₃(0.413) = **4.647006**
- Интервал [0.23, 1.40]: P₃(0.413) = **5.973281**

## Преимущества метода Ньютона

1. **Эффективность:** Легко добавлять новые узлы без пересчёта всего многочлена
2. **Оценка погрешности:** Встроенная оценка через разделённые разности
3. **Численная устойчивость:** Часто более стабилен, чем метод Лагранжа
4. **Результаты идентичны:** При одинаковых узлах даёт те же значения, что и Лагранж

Видно, что разные интервалы дают **существенно разные результаты** интерполяции!
