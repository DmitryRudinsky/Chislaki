# Лабораторная работа 3.3 - Кубический сплайн

## Описание

Интерполяция функции с помощью естественного кубического сплайна дефекта 1.

## Данные

- **Точка интерполяции:** x* = 9.625
- **Узлы:** 11 точек
- **Вариант:** 29
- **Метод:** Естественный кубический сплайн (граничные условия: S''(x₀) = S''(xₙ) = 0)

## Запуск

### Графический интерфейс (Electron)
```bash
npm install
npm start
```

### Консольные вычисления
```bash
npm run calc
```

## Функции

- `buildNaturalCubicSpline()` - построение естественного кубического сплайна
- `solveTridiagonal()` - решение трёхдиагональной СЛАУ
- `evaluateSpline()` - вычисление значения сплайна
- `findSplineSegment()` - поиск сегмента сплайна
- `verifySplineAtNodes()` - проверка интерполяции

## Результаты

Программа выводит:
- Коэффициенты сплайна для каждого сегмента (a, b, c, d)
- Значение в точке x*
- Формулу сплайна на отрезке, содержащем x*
- Проверку интерполяции во всех узловых точках
- Графики сплайна и его сегментов

## Математика

Для каждого сегмента [xᵢ, xᵢ₊₁] сплайн имеет вид:

S₃(x) = aᵢ + bᵢ(x - xᵢ) + cᵢ(x - xᵢ)² + dᵢ(x - xᵢ)³

где коэффициенты находятся из условий:
- Интерполяция в узлах
- Непрерывность первой и второй производных
- Естественные граничные условия
