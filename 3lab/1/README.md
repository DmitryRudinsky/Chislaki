# Лабораторная работа 3.1 - Интерполяция Лагранжа

## Описание

Интерполяция функции с помощью многочленов Лагранжа 2-й и 3-й степени.

## Данные

- **Точка интерполяции:** x* = 0.413
- **Узлы:** 9 точек
- **Степени многочленов:** 2 и 3

## Запуск

### Графический интерфейс (Electron)
```bash
npm install
npm start
```

### Консольные вычисления
```bash
npm run calc
```

## Структура

```
3lab/1/
├── lagrange.js       # Реализация метода Лагранжа
├── graph.js          # Визуализация с Chart.js
├── index.html        # Веб-интерфейс
├── electron-main.js  # Electron entry point
├── package.json      # Конфигурация
└── README.md         # Документация
```

## Функции

- `chooseNearestNodes()` - выбор ближайших узлов
- `lagrangeCoefficients()` - вычисление коэффициентов
- `evaluatePolynomial()` - вычисление значения многочлена
- `verifyAtAllNodes()` - проверка интерполяции

## Результаты

Программа выводит:
- Многочлены Лагранжа для степеней 2 и 3
- Значения в точке x*
- Проверку интерполяции в узловых точках
- Графики интерполяции
