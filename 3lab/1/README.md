# Лабораторная работа 3.1 - Интерполяция Лагранжа

## Описание

Интерполяция функции с помощью многочленов Лагранжа 2-й и 3-й степени.

**ОБНОВЛЕНИЕ:** Теперь программа рассматривает **все возможные последовательные интервалы**, а не только ближайшие точки!

## Данные

- **Точка интерполяции:** x* = 0.413
- **Узлы:** 9 точек
- **Степени многочленов:** 2 и 3
- **Интервалов для P₂:** 7 (из них 2 содержат x*)
- **Интервалов для P₃:** 6 (из них 3 содержат x*)

## Функции

- `chooseNearestNodes()` - выбор ближайших узлов (старый подход)
- `getAllConsecutiveIntervals()` - **НОВАЯ**: генерация всех последовательных интервалов
- `isPointInInterval()` - **НОВАЯ**: проверка принадлежности точки интервалу
- `lagrangeCoefficients()` - вычисление коэффициентов
- `evaluatePolynomial()` - вычисление значения многочлена
- `verifyAtAllNodes()` - проверка интерполяции

## Что показывает программа

### Графики
- **Отдельный график для каждого интервала** (всего 13 графиков)
- Интервалы, содержащие x*, выделены **зелёной рамкой**
- На каждом графике:
  - Многочлен Лагранжа на всём диапазоне
  - Все исходные точки (серые)
  - Узлы интерполяции для данного интервала (зелёные треугольники)
  - Точка x* = 0.413 (звезда)

### Таблица сравнения
- Все интервалы с их границами
- Значения P(x*) для каждого интервала
- Выделение интервалов, содержащих x*

## Ключевые результаты

### Многочлен 2-й степени
- Интервал [-0.16, 0.62]: P₂(0.413) = **4.137985**
- Интервал [0.23, 1.01]: P₂(0.413) = **5.177347**

### Многочлен 3-й степени
- Интервал [-0.55, 0.62]: P₃(0.413) = **3.532009**
- Интервал [-0.16, 1.01]: P₃(0.413) = **4.647006**
- Интервал [0.23, 1.40]: P₃(0.413) = **5.973281**

Видно, что разные интервалы дают **существенно разные результаты** интерполяции!
