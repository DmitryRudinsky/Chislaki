# Лабораторная работа 3.6 - Численное интегрирование

## Описание

Лабораторная работа по численным методам интегрирования. Реализованы четыре метода вычисления определенного интеграла с оценкой погрешностей и уточнением по методу Рунге-Ромберга.

## Задание (Вариант 29)

Вычислить определенный интеграл:

```
F = ∫ [tg(x/2) + ln(x+1)] / [cos²(x-1) + 1] dx
    -0.5 до 1.5
```

### Требования

1. Вычислить интеграл по формулам:
   - Средних прямоугольников
   - Трапеций
   - Симпсона
   - Эйлера

2. Выбрать шаг интегрирования так, чтобы можно было применить все методы

3. Оценить погрешности методов интегрирования

4. Уменьшить шаг в два раза и повторить вычисления

5. Уточнить значения по методу Рунге-Ромберга

## Реализованные методы

### 1. Метод средних прямоугольников

Вычисляет интеграл через значения функции в середине каждого отрезка разбиения.

**Формула:**
```
∫f(x)dx ≈ h · Σf(x_{i+1/2})
```

**Порядок точности:** O(h²)

### 2. Метод трапеций

Аппроксимирует функцию линейными отрезками на каждом подынтервале.

**Формула:**
```
∫f(x)dx ≈ h · [f(a)/2 + Σf(x_i) + f(b)/2]
```

**Порядок точности:** O(h²)

### 3. Метод Симпсона

Аппроксимирует функцию параболами на каждых двух отрезках разбиения.

**Формула:**
```
∫f(x)dx ≈ (h/3) · [f(a) + 4·Σf(x_{2i-1}) + 2·Σf(x_{2i}) + f(b)]
```

**Порядок точности:** O(h⁴)

**Требование:** Число отрезков должно быть четным.

### 4. Метод Эйлера

Улучшенный метод трапеций с учетом производных на концах интервала.

**Формула:**
```
∫f(x)dx ≈ I_трапеций + (h²/12)·[f'(a) - f'(b)]
```

**Порядок точности:** O(h⁴)

### 5. Метод Рунге-Ромберга

Используется для уточнения результатов путем экстраполяции двух вычислений с разными шагами.

**Формула:**
```
I* = I_{h/2} + (I_{h/2} - I_h) / (k^p - 1)
```

где:
- `I_h` - значение интеграла с шагом h
- `I_{h/2}` - значение интеграла с шагом h/2
- `k = 2` - кратность уменьшения шага
- `p` - порядок точности метода

## Результаты

### Параметры вычислений

- **Интервал:** [-0.5, 1.5]
- **Начальный шаг h₁:** 0.2
- **Уменьшенный шаг h₂:** 0.1

### Эталонное значение

Для оценки погрешностей используется "эталонное" значение, вычисленное методом Симпсона с очень мелким шагом (h = 0.00001).

### Визуализация

Графический интерфейс включает:

1. **График подынтегральной функции** - визуализация функции на заданном интервале

2. **Сравнение методов для h₁** - столбчатая диаграмма значений интеграла разными методами

3. **Сравнение методов для h₂** - то же для уменьшенного шага

4. **График погрешностей** - сравнение погрешностей при разных шагах (логарифмическая шкала)

5. **Эффект Рунге-Ромберга** - демонстрация улучшения точности после уточнения

6. **Таблица результатов** - детальные численные результаты для всех методов

## Выводы

1. **Порядок точности методов:**
   - Метод Симпсона и Эйлера имеют наивысшую точность (O(h⁴))
   - Методы прямоугольников и трапеций менее точны (O(h²))

2. **Уменьшение шага:**
   - При уменьшении шага в 2 раза погрешность уменьшается в:
     - ~4 раза для методов 2-го порядка
     - ~16 раз для методов 4-го порядка

3. **Метод Рунге-Ромберга:**
   - Позволяет значительно повысить точность без дополнительных вычислений функции
   - Особенно эффективен для методов низкого порядка

4. **Выбор метода:**
   - Для быстрых вычислений: метод трапеций или средних прямоугольников
   - Для высокой точности: метод Симпсона или Эйлера
   - Для оптимального соотношения: метод Симпсона с уточнением Рунге-Ромберга
