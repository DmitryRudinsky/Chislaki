Условие задачи

Интегрируемая функция
f(x) = [tg(x/2) + ln(x+1)] / [cos²(x-1) + 1]
Интервал интегрирования
[a, b] = [-0.5, 1.5]
Начальный шаг
h₁ = 0.2
Уменьшенный шаг
h₂ = 0.1 (h₁/2)

Метод средних прямоугольников

Вычисление интеграла через значения функции в середине каждого отрезка разбиения. Погрешность O(h²).

Метод трапеций

Аппроксимация функции линейными отрезками. Погрешность O(h²).

Метод Симпсона

Аппроксимация функции параболами на каждых двух отрезках. Погрешность O(h⁴).

Метод Эйлера

Улучшенный метод трапеций с учетом производных на концах интервала. Погрешность O(h⁴).

Эффект уточнения по методу Рунге-Ромберга

Метод Рунге-Ромберга: Используется для уточнения результатов путем экстраполяции двух вычислений с разными шагами. Формула: I* = Ih/2 + (Ih/2 - Ih) / (kp - 1), где k=2, p - порядок метода.
